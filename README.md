# Artline Motors Lots Bot

Telegram-бот для менеджеров по продажам, которые руками заносят лоты автомобилей, добавляют фото и ставят публикации в Telegram-канал в отложку.

## Что умеет MVP

- вход по внутреннему номеру менеджера;
- выбор источника автомобиля: `США`, `ОАЭ`, `ЮЖНАЯ КОРЕЯ`, `КИТАЙ`, `ЕВРОПА`;
- создание нового лота с ручным вводом стоимости и расходов;
- отдельные поля под шаблон поста: пробег и привод;
- автоматический расчет пошлины, таможенного оформления и утильсбора по логике из калькулятора;
- сохранение фотографий через `Telegram file_id`, без отдельного фотохранилища;
- отложенная публикация в канал;
- админский просмотр очереди и ручное управление публикацией;
- статистика по менеджерам за период с `16` числа по `16` число.
- публикация с кликабельными ссылками `Telegram` / `WhatsApp` / сайт через HTML-разметку Telegram;
- хештеги по бренду, стране и бюджету с округлением бюджета вверх до миллиона.

## Почему фото лучше хранить как `file_id`

Когда менеджер отправляет фото в бота, Telegram возвращает `file_id`. Его можно сохранить в базе и потом публиковать в канал без скачивания файла на диск и без отдельного S3/Google Drive.

Плюсы:

- не нужен отдельный сервер для изображений;
- повторная публикация работает быстрее;
- на Bothost и похожих хостингах меньше рисков с файловой системой.

## Хранилище

Основная база MVP хранится в `data/db.json`.

Это удобнее Excel для:

- очереди публикаций;
- статусов лотов;
- отложки;
- статистики по менеджерам;
- сохранения `file_id` фотографий.

Если позже понадобится, из JSON легко сделать экспорт в Excel или Google Sheets.

## Настройка

```bash
cp .env.example .env
```

Переменные:

- `BOT_TOKEN` — токен Telegram-бота.
- `CHANNEL_ID` — id канала или `@username` канала для публикаций, например `-100...`.
- `MANAGERS` — список менеджеров в формате `101:Иван,102:Павел`.
- `ADMIN_CODES` — номера, у которых есть доступ к админке.
- `DEFAULT_LAB_RUB` — лаборатория по умолчанию.
- `DEFAULT_BROKER_USD` — брокер РФ по умолчанию.

## Запуск

```bash
npm test
BOT_TOKEN=123456:token npm start
```

## Управление в боте

Менеджер:

- `Новый лот`
- `Моя статистика`
- `Мои лоты`

Админ дополнительно:

- `Очередь публикаций`
- `Статистика менеджеров`

## Шаблон публикации

Пост в канал собирается в формате:

- название авто;
- год выпуска, пробег, двигатель, привод;
- итоговая цена под ключ до выбранного города;
- хештеги бренда, бюджета и страны;
- кликабельные ссылки на Telegram, WhatsApp и сайт.

## Логика статистики

Период считается не по календарному месяцу, а от `16` числа текущего периода до `16` числа следующего.

Пример:

- если сейчас `20.06.2026`, окно статистики: `16.06.2026 00:00` — `16.07.2026 00:00`;
- если сейчас `10.06.2026`, окно статистики: `16.05.2026 00:00` — `16.06.2026 00:00`.
